garlic
A new language of time displays the current time by the degrees of rotation of garlic cloves.
watch interface animation tool random animation player
I created the cloves in Maya and manipulated them using Three.js.
This project began as a simple idea of allowing the garlic to decay over time and associating the state of decay with the current time of day ..
But playing around with the 3 dimensional shapes and code led to unexpected visuals that i fell in love with.
